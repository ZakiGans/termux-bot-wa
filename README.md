## Install
Follow The Steps Below!

## For Termux
```bash
> termux-setup-storage
(after that tap on permission)
> apt install git -y
> git clone https://github.com/ZakiGans/termux-bot-wa
> cd termux-bot-wa
> bash install.sh
```

### Starting Bot

```bash
> npm start/node index.js
```
